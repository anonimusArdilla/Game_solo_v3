# Lantern Drift

Hybrid-casual, ads-only mobile arcade built with Flutter + Flame.

## Quick Start

```bash
cp .env.example .env
/workspace/.tool/flutter/bin/flutter pub get
/workspace/.tool/flutter/bin/flutter gen-l10n
/workspace/.tool/flutter/bin/flutter test
/workspace/.tool/flutter/bin/flutter analyze
/workspace/.tool/flutter/bin/flutter run
```

## Builds

```bash
/workspace/.tool/flutter/bin/flutter build apk --release
/workspace/.tool/flutter/bin/flutter build appbundle --release
```
